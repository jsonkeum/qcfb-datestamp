Datestamp Microservice API

by QuanchiFootball


This is a simple API that will take an URL date paramater and return a JSON object containing the date in both Unix time format and natural language string format.


Example

Possible URL Inputs :

NATURAL LANGUAGE FORMAT:

https://qcfb-datestamp.glitch.me/august 29, 1978
https://qcfb-datestamp.glitch.me/august291978
https://qcfb-datestamp.glitch.me/august29,1978
https://qcfb-datestamp.glitch.me/8 29 1978

UNIX TIME FORMAT:

https://qcfb-datestamp.glitch.me/273196800

All return :

{"unix":273196800,"natural":"August 29, 1978"}

Spaces and commas can be omitted when using a literal month name e.g. august, may. If an incorrect date parameter is entered, the API will return the object with null values.

Note that the API only returns the natural language date and not the time. Therefore a range of Unix inputs will result in the same natural language output.


Unix vs Natural Language

Use spaces to delineate between month, day and year when using a numerical month. Failure to do so may result in an incorrect Unix time output:

//A natural language query for August 29, 1978 - numerical with no spaces
https://qcfb-datestamp.glitch.me/8291978

Because the API takes in both Unix and natural language, it will take this request as a Unix input, resulting in a different date:

{"unix":8292017,"natural":"April 6, 1970"}
